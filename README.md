# Weather App using OpenWeatherMap API

A responsive Weather App built with HTML, CSS, and JavaScript that fetches real-time weather data using the OpenWeatherMap API and displays temperature, humidity, and weather conditions in a modern user interface.

## Features

* Search weather by city name
* Real-time weather information
* Temperature display in Celsius
* Humidity information
* Weather condition display
* Dynamic weather icons
* Responsive design for mobile and desktop
* Clean and modern Glassmorphism UI
* Error handling for invalid city names

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Fetch API
* OpenWeatherMap API

## How to Run

1. Clone or download this repository.
2. Open the project folder.
3. Create a free account on OpenWeatherMap and generate an API key.
4. Open the `script.js` file.
5. Replace:

```javascript
const apiKey = "YOUR_API_KEY";
```

with your actual API key:

```javascript
const apiKey = "your_api_key_here";
```

6. Save the file.
7. Open `index.html` in your web browser.
8. Enter a city name and click the search button to view the weather details.
